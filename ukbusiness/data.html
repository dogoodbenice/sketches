What did I learn when making this?

Aside from the main points, I learnt:

  • Making charts responsive for a novice like me is fairly difficult. There's lots of room to improve so apologies if you read this on a mobile device and didn't have a great time.
  • Data is, thankfully, everywhere. Once you start searching it makes me glad the UK publishes and shares so much data.
  • It takes time to understand the complexity. Whilst data is accessible the items behind the figures take a little time and context to unpick.
  • Charting libraries are really powerful. Thanks to all the creators and maintainers of those libraries creating and iterating on these charts using the data is really fast.

That's it, thanks for reading. If you have any feedback, requests for future articles or thoughts please do share it with me.
